#  Cloud-Based File Sharing System Using Flask

A simple and secure web-based file sharing application built using Python Flask. This system allows users to register, login, upload files, download files, and delete files through a clean and user-friendly dashboard interface.

---

##  Features

- User Registration and Login system  
- Secure authentication using Flask sessions  
- Upload files easily  
- Download uploaded files  
- Delete files from dashboard  
- Simple and clean UI  

---

##  Technologies Used

- HTML
- CSS
- Bootstrap

##  How to Run

1. Install Flask  
python -m pip install flask  

2. Run project  
python app.py  

3. Open browser  
http://127.0.0.1:5000
